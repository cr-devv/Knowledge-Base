## Comandos básicos de git

Preparar los archivos para subirse 
`git add .`

Tomar la foto (commit)
```git commit -m "Breve descripción de cambios"```

Enviar a la nube
```git push```

Verificar que archivos se van a subir antes del git add
```git status```

Bajar desde la nube cambios de github a mi pc, usarlo siempre que haya tocado algo en github fuera de mi pc 
```git pull```
